# Zerodha Clone 🚀

A fully responsive Zerodha landing page clone built using React and Bootstrap.

## 🔥 Features

* Responsive landing page
* Clean UI similar to Zerodha
* Reusable React components
* Bootstrap-based layout
* Image assets and proper grid structure

## 🛠️ Tech Stack

* React.js
* Bootstrap 5
* HTML5
* CSS3
* Vite

## 📂 Project Structure

frontend/
│── src/
│── public/
│── images/
│── components/
│── index.html

## 🚀 Installation & Setup

Clone the repository:

git clone https://github.com/your-username/zerodha-clone.git

Go to project folder:

cd frontend

Install dependencies:

npm install

Start the development server:

npm run dev

Open in browser:
http://localhost:5173
